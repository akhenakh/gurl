Gurl

A better curl in Go language.

Main Features

  • Static binary, thanks to Go
  • Submit and indent JSON format
  • Forms
  • Custom Headers
  • Load balancers tester

Usage

Synopsis:

gurl [flags] [METHOD] URL [ITEM [ITEM]]

See also gurl -help.

Examples

Submitting forms as x-www-form-urlencoded.

gurl POST example.org name=bob
HTTP/1.1 200 OK
Content-Length: 0
Connection: keep-alive
Content-Type: application/json
Date: Wed, 28 Nov 2012 09:31:03 GMT
Server: gunicorn/0.13.4

{}

See the request that is being sent.

gurl -verbose GET myapi.org
POST / HTTP/1.1
Host: myapi.org
Accept: */*
Accept-Encoding: identity, deflate, compress, gzip
Content-Type: application/json
User-Agent: Gurl

HTTP/1.1 200 OK
Content-Length: 0
Connection: keep-alive
Content-Type: application/json
Date: Wed, 28 Nov 2012 09:31:03 GMT
Server: gunicorn/0.13.4

{}

Submitting as application/json

gurl -json POST httpbin.org/post name=bob

Change a header use ":" serator

gurl GET httpbin.org X-Id:42

Direct connection to a specified host, makes testing website easier, especially load balanced ones

gurl -verbose -server=myfront1 GET mywebsite.com 
GET / HTTP/1.1
Host: mywebsite.com
Accept: */*
Accept-Encoding: identity, deflate, compress, gzip
User-Agent: Gurl

HTTP/1.1 200 OK
Accept-Ranges: bytes
Connection: keep-alive
Content-Type: text/html
Date: Wed, 28 Nov 2012 09:36:56 GMT
Last-Modified: Mon, 08 May 2006 19:39:45 GMT
Server: nginx/1.2.5
